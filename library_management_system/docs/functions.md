| F-S06  | 資料詳細取得                 | 資料の詳細情報・貸出状況・予約状況を確認する                       | UC-06             |
| F-S07  | 資料予約（自校蔵書）         | 利用可能な自校資料に予約を入れる                                   | UC-07             |
| F-S08  | 順番待ち登録                 | 貸出中資料にキュー登録する                                         | UC-08             |
| F-S09  | 予約キャンセル               | 自校資料・図書館間貸出の予約をキャンセルする                       | UC-09             |

| F-S11  | 資料貸出                     | SCR-16（貸出処理画面）でバーコードをスキャンし自身の貸出を確定する | UC-11             |
| F-S12  | 資料返却                     | SCR-17（返却処理画面）でバーコードをスキャンし自身の返却を確定する | UC-12             |
| F-S13  | 自分の貸出一覧確認           | 貸出中・貸出履歴を確認する                                         | UC-13             |

| F-S16  | 自分の図書館間貸出一覧確認   | 図書館間貸出の状況を確認する                                       | UC-19             |
| F-S17  | マイページ閲覧               | 貸出中・予約中・順番待ち・図書館間貸出の状況をまとめて確認         | UC-10,UC-13,UC-19 |

### システム自動処理

| ------- | ------------------------ | --------------------------------------------------------- | ----------- |
| F-SYS01 | 予約準備通知メール       | 予約資料が準備できた際にメール送信                        | UC-07,UC-21 |

| F-SYS06 | 延滞制限チェック         | 貸出・予約時に延滞中資料の有無を検証（API・フロント両方） | -           |
