# ユースケース：検索・閲覧

---

### UC-33：資料検索
- **アクター**：生徒、司書、管理者
- **目的**：全校の蔵書を横断的に検索する
- **前提条件**：
  - システムにアクセス可能である（ログイン有無は問わない場合もあり）
- **基本フロー**：
  1. 検索バーにキーワードを入力（タイトル・著者・ISBN・NDCなど）
  2. 検索を実行
  3. システムが該当資料を検索
  4. 検索結果一覧を表示
     - タイトル
     - 著者
     - 所蔵校
     - 貸出状況（貸出可／貸出中／予約あり）
- **代替フロー**：
  - 該当結果がない場合：該当なしメッセージ表示
- **例外**：
  - システムエラー時：エラーメッセージ表示
- **関連API**：
  - GET /books?query=xxx
- **関連画面**：
  - SCR-05（検索結果一覧）

---

### UC-34：資料詳細閲覧
- **アクター**：生徒、司書、管理者
- **目的**：資料の詳細情報および利用状況を確認する
- **前提条件**：
  - 対象資料が存在する
- **基本フロー**：
  1. 検索結果から資料を選択
  2. 詳細ページを表示
     - タイトル、著者、ISBN、NDC分類
     - 所蔵校一覧
     - 貸出状況
     - 予約状況（待ち人数など）
  3. 利用可能な操作を表示（予約・順番待ち等）
- **例外**：
  - 資料が削除済み・非公開の場合：閲覧不可
- **関連API**：
  - GET /books/{id}
- **関連画面**：
  - SCR-06（資料詳細）

---

### UC-35：検索フィルタ・並び替え
- **アクター**：生徒、司書、管理者
- **目的**：検索結果を条件で絞り込み、目的の資料を見つけやすくする
- **前提条件**：
  - 検索結果が存在する
- **基本フロー**：
  1. フィルタ条件を指定
     - 所蔵校
     - 貸出状況
     - NDC分類
  2. 並び替え条件を選択
     - タイトル順
     - 著者順
     - 登録日順
  3. 条件に応じて結果を再表示
- **例外**：
  - 条件に一致する結果がない場合：該当なし表示
- **関連API**：
  - GET /books?query=xxx&filters=...&sort=...
- **関連画面**：
  - SCR-05（検索結果一覧）
