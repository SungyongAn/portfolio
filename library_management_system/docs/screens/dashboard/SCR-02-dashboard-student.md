
# SCR-02：ダッシュボード（生徒・図書委員）
**表示**
- 貸出中資料一覧（返却期限・残日数）
- 返却期限が近い資料の警告
- 予約中・順番待ち資料一覧
- 図書館間貸出の状況

**表示（図書委員のみ）**
- 代理操作メニュー（`is_committee: true`の場合のみ表示）
  - 対象生徒検索・選択欄（自校生徒のみ）
  - 選択生徒の貸出中・予約状況

**遷移先**
- 資料検索（SCR-05）
- 貸出処理（SCR-16）
- 返却処理（SCR-17）
- マイページ（SCR-07）
- 貸出管理（SCR-14）（図書委員のみ）
- パスワード変更（SCR-08）
