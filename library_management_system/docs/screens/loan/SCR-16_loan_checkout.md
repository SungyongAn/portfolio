# SCR-16：貸出処理

**対象ロール**：student / committee / librarian

**入力**

- 資料バーコードをスキャン（またはID手入力）

**表示**

- 資料情報（タイトル・著者・所蔵校）
- 貸出可否（上限・延滞・他の予約者有無を確認後に表示）
- 返却期限（貸出確定後・貸出日から2週間）

**操作（committee / librarian のみ表示）**

- 対象生徒検索・選択欄（自校生徒のみ）
- 選択生徒の貸出冊数・延滞状況を表示
- 貸出上限・延滞制限は代理操作時も同様に適用

**操作制限（フロント表示・バックエンドでも検証）**

- 貸出上限（5冊）超過時 → 「貸出上限に達しています」を表示し貸出不可
- 延滞中資料あり → 「延滞中の資料があります」を表示し貸出不可
- 他の予約者あり → 「他に予約者がいるため貸出できません」を表示し貸出不可

**遷移先**

- 貸出確定後 → 同画面リセット（連続処理対応）
