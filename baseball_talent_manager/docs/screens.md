# 画面一覧

## 画面一覧サマリ（画面数・遷移の概要）
- 画面数：5（viewsのルーティング単位）
- 簡易画面の概要
```
src/
├── components/
│   ├── MemberManagement.vue（部員の管理画面、コーチ / 監督）
│   ├── MemberCreate.vue（部員作成、コーチ / 監督）
│   ├── MemberRetire.vue（退部・引退処理、コーチ / 監督）
│   ├── MeasurementResultReview.vue（測定結果の確認/承認画面、部員 / コーチ）
│   ├── MeasurementResultList.vue（測定記録の閲覧、全アカウント）
│   ├── MeasurementResultGraph.vue（測定記録のグラフ閲覧、全アカウント）※課題2
│   └── AppHeader.vue（ログイン後の共有ヘッダー）
└── views/
    ├── LoginView.vue（ログイン画面）
    ├── shared/
    │   └── DashboardView.vue（部員 / コーチ / 監督）
    └── manager/
        ├── DashboardView.vue（マネージャー）
        ├── MeasurementResultSubmit.vue（測定結果の入力/承認依頼）
        └── MeasurementStatusList.vue（承認依頼中のステータス確認）
```

### manager/MeasurementResultSubmit.vue（測定結果の入力/承認依頼）
### ■ 課題1
- **対象ロール**：マネージャー
- **主な表示項目**
  - 部員名
  - 学年
  - 走力（2種）：50m走（sec）/ ベースランニング（sec）
  - 肩力（2種）：遠投（m）/ ストレート球速（km/h）
  - 打力（2種）：打球速度（km/h）/ スイング速度（km/h）
  - 筋力（2種）：ベンチプレス（kg）/ スクワット（kg）
  - 保存ボタン
  - 承認依頼状況の確認ボタン
  - 保存ボタンを押した後、保存内容と承認依頼ボタン
- **主な入力項目**
  - 測定日（カレンダーから選択 or 手入力）
  - 部員名
  - 学年
  - 走力（2種）：50m走（sec）/ ベースランニング（sec）
  - 肩力（2種）：遠投（m）/ ストレート球速（km/h）
  - 打力（2種）：打球速度（km/h）/ スイング速度（km/h）
  - 筋力（2種）：ベンチプレス（kg）/ スクワット（kg）
- **遷移先**
  - 保存ボタン → 送信される対象部員情報の確認画面を表示後、承認依頼を送信
  - MeasurementStatusList.vue（承認依頼中のステータス確認、承認依頼状況の確認ボタン）

### ■ 課題2
- 複数部員一括入力
- ページネーション（10人単位）
- 送信済 / 未送信管理UI

### MeasurementResultReview.vue（測定結果の確認/承認画面）
### ■ 課題1
- **対象ロール**：部員 / コーチ
- **主な表示項目**
  - 測定結果の一覧表示
    - 部員：自身の未承認測定結果のみ表示
    - コーチ：全部員の部員承認済み・コーチ未承認の測定結果を表示
  - 承認ボタン
  - 否認ボタン
- **主な入力項目**：なし
- **遷移先**
  - 承認・否認ボタン押下後、同画面内に留まり未処理の一覧のみ表示、未処理がなければ項目名のみ表示

### ■ 課題2
- 承認履歴表示
- 承認コメント機能
