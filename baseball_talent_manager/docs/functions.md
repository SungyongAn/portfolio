# PoC機能一覧

## 認証機能

| 機能名 | 優先度 | 対象ロール | 備考 |
|---|---|---|---|
| ログイン | MUST | 全ユーザー | メール＋パスワード認証 |
| ログアウト | MUST | 全ユーザー | HttpOnly Cookie削除 |
| トークン管理 | MUST | 全ユーザー | アクセストークン：メモリ / リフレッシュトークン：Cookie |

## 測定記録機能

| 機能名 | 優先度 | 対象ロール | 備考 |
|---|---|---|---|
| 測定結果の入力 | MUST | マネージャー | ページネーション・一括送信 / 測定日はカレンダーから選択 |
| 送信前確認画面の表示 | MUST | マネージャー | 送信済み部員／未送信部員の一覧を送信前に表示 |
| 承認フローの発行 | MUST | マネージャー | 全項目入力済み部員のみ発行 |
| 測定結果の再入力 | MUST | マネージャー | 否認ステータスを測定結果一覧で確認後に再入力・再発行 |
| 測定結果の承認 | MUST | 部員 | 個別承認のみ / 値の修正不可 |
| 測定結果の否認 | MUST | 部員 | 個別否認のみ / 否認理由の記録はなし |
| 測定結果の最終承認 | MUST | コーチ | 個別承認のみ / 一括承認不可 |
| 測定結果の最終否認 | MUST | コーチ | 個別否認のみ / 否認理由の記録はなし |
| 測定結果一覧の確認 | MUST | マネージャー | 否認ステータスの確認・再入力の起点となる画面 |


## 閲覧・可視化機能

| 機能名 | 優先度 | 対象ロール | 備考 |
|---|---|---|---|
| 自身の測定記録の閲覧 | MUST | 部員 | 承認済みの自身の全測定記録を一覧表示 / 絞り込みなし |
| 全部員の測定記録の閲覧 | MUST | コーチ / 監督 | 承認済みの全部員の測定記録を閲覧可能 / 検索・ソートあり |
| 測定記録のグラフ表示 | WANT | 部員 | 専用ページで項目混合・項目別など表示方法を任意選択可能 |

## 部員管理機能

| 機能名 | 優先度 | 対象ロール | 備考 |
|---|---|---|---|
| 部員一覧の参照 | MUST | マネージャー / コーチ / 監督 | マネージャーは参照のみ |
| 部員作成 | MUST | コーチ / 監督 | 登録項目：メールアドレス・生徒名・学年・パスワード / パスワードはコーチ・監督が初期設定 |
| 退部処理 | MUST | コーチ / 監督 | 確認画面あり / ソフトデリートで退部部員として管理 |
| 引退処理 | MUST | コーチ / 監督 | 確認画面あり / 3年生一括引退処理可能 / ソフトデリートで引退部員として管理 |

## 共通UI機能

| 機能名 | 優先度 | 対象ロール | 備考 |
|---|---|---|---|
| 共通ヘッダー表示 | MUST | 全ユーザー | ユーザー名・ロール・ログアウトボタンを常時表示 |
| ロール別画面振り分け | MUST | 全ユーザー | ログイン後にロールに応じた画面へ遷移 |
| 誤操作防止確認ダイアログ | MUST | コーチ / 監督 | 退部・引退処理時に確認画面を表示 |

## 対象外機能（PoC割り切り）

| 機能名 | 理由 |
|---|---|
| MFA | 未経験技術のため将来対応として設計メモのみ |
| 否認理由の記録 | PoCスコープ外 |
| パスワード初回強制変更 | PoCスコープ外 / セキュリティ上の割り切りとしてドキュメントに明記 |
| リフレッシュトークンのDB管理 | PoCスコープ外 / Cookie管理のみで対応 |
| チーム全体の傾向可視化 | 課題2スコープ |
