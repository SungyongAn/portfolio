# PoC機能一覧

## 認証機能

| 機能名 | 優先度 | 対象ロール | 備考 |
|---|---|---|---|
| ログイン | MUST | 全ユーザー | メール＋パスワード認証 |
| ログアウト | MUST | 全ユーザー | HttpOnly Cookie削除 |
| トークン管理 | MUST | 全ユーザー | アクセストークン：メモリ / リフレッシュトークン：Cookie |

## 測定記録機能

| 機能名 | 優先度 | 対象ロール | 備考 |
|---|---|---|---|
| 測定結果の入力 | MUST | マネージャー | 1件ずつ入力 / 測定日はカレンダーから選択 |
| 送信前確認モーダルの表示 | MUST | マネージャー | 入力内容を確認後に承認依頼を送信 |
| 承認フローの発行 | MUST | マネージャー | 全項目入力済みの場合のみ発行可能 |
| 重複登録防止 | MUST | マネージャー | 同一部員・同一計測日の重複チェック / rejected レコードは上書き更新 |
| 測定結果の再入力 | MUST | マネージャー | 否認ステータスを確認後に再入力・再発行 |
| 測定結果の承認 | MUST | 部員 | 個別承認のみ / 値の修正不可 |
| 測定結果の否認 | MUST | 部員 | 個別否認のみ / 否認理由の記録はなし |
| 測定結果の最終承認 | MUST | コーチ | 個別承認のみ / 一括承認不可 |
| 測定結果の最終否認 | MUST | コーチ | 個別否認のみ / 否認理由の記録はなし |
| 承認フローステータス確認 | MUST | マネージャー | 否認・部員承認待ち・コーチ承認待ちの件数を確認 |
| 承認済みレコードの確認済み管理 | MUST | マネージャー | 確認ボタンで承認済みレコードを非表示 / DB永続化（manager_confirmed） |

## 閲覧・可視化機能

| 機能名 | 優先度 | 対象ロール | 備考 |
|---|---|---|---|
| 自身の測定記録の閲覧 | MUST | 部員 | 承認済みの自身の全測定記録を一覧表示 |
| 全部員の測定記録の閲覧 | MUST | コーチ / 監督 / マネージャー | 承認済みの全部員の測定記録を閲覧可能 |
| 測定記録のソート・絞り込み | MUST | 全ユーザー | 部員名・学年・計測日・各測定項目でソート / 絞り込み対応 |
| ページネーション | MUST | 全ユーザー | 表示件数切り替え（10件・20件・50件）/ URLクエリパラメータ同期 |
| 成長推移グラフ（折れ線） | WANT | 全ユーザー | 項目別チーム平均・個人推移を時系列表示 / ECharts使用 |
| 能力比較グラフ（レーダー） | WANT | コーチ / 監督 | 部員間の能力比較・チーム平均比較 / ECharts使用 |
| チーム内順位表 | WANT | 全ユーザー | 測定項目別の順位表・自分の位置づけ確認 |

## 通知機能

| 機能名 | 優先度 | 対象ロール | 備考 |
|---|---|---|---|
| ダッシュボード通知サマリー | WANT | マネージャー / 部員 / コーチ | 承認状況をダッシュボードに集計表示 |
| WebSocketリアルタイム通知 | WANT | 全ユーザー | 承認フローのステータス変化をリアルタイムに通知 / 画面遷移不要 |

## 部員管理機能

| 機能名 | 優先度 | 対象ロール | 備考 |
|---|---|---|---|
| 部員一覧の参照 | MUST | マネージャー / コーチ / 監督 | マネージャーは参照のみ |
| 部員作成 | MUST | コーチ / 監督 | 登録項目：メールアドレス・生徒名・学年・パスワード / パスワードはコーチ・監督が初期設定 |
| 退部処理 | MUST | コーチ / 監督 | 確認画面あり / ソフトデリートで退部部員として管理 |
| 引退処理 | MUST | コーチ / 監督 | 確認画面あり / ソフトデリートで引退部員として管理 |

## 共通UI機能

| 機能名 | 優先度 | 対象ロール | 備考 |
|---|---|---|---|
| 共通ヘッダー表示 | MUST | 全ユーザー | ユーザー名・ロール・ログアウトボタンを常時表示 |
| ロール別画面振り分け | MUST | 全ユーザー | ログイン後にロールに応じた画面へ遷移 |
| 誤操作防止確認ダイアログ | MUST | コーチ / 監督 | 退部・引退処理時に確認画面を表示 |

## 対象外機能（PoC割り切り）

| 機能名 | 理由 |
|---|---|
| MFA | 未経験技術のため将来対応として設計メモのみ |
| 否認理由の記録 | PoCスコープ外 |
| パスワード初回強制変更 | PoCスコープ外 / セキュリティ上の割り切りとしてドキュメントに明記 |
| リフレッシュトークンのDB管理 | PoCスコープ外 / Cookie管理のみで対応 |
| 複数ワーカー対応（Redis Pub/Sub） | PoCスコープ外 / シングルワーカー構成のため不要 |
| 3年生一括引退処理 | PoCスコープ外 / 将来の拡張として検討 |
| 一括測定入力 | PoCスコープ外 / 将来の拡張として検討 |
