# 画面設計

> 課題１の必須機能をもとに、画面一覧・遷移・入力項目・表示項目を整理します。

---

## ■ 画面一覧

| 画面ID | 画面名 | 対象ロール | 関連機能ID |
|--------|--------|------------|------------|
| S-AUTH-01 | ログイン画面 | 全ロール | F-AUTH-01 |
| S-CMN-01 | ダッシュボード（トップ） | 全ロール | F-APR-03 / F-NTF-01 |
| S-APP-01 | 案件一覧画面 | 全ロール | F-APP-02 / F-PRG-01 / F-PRG-02 |
| S-APP-02 | 案件申請画面 | APPLICANT | F-APP-01 |
| S-APP-03 | 案件詳細画面 | 全ロール | F-APP-03 / F-TSK-01 / F-TSK-02 |
| S-APR-01 | 承認・却下画面 | DEPT_MANAGER / HQ_MANAGER | F-APR-01 / F-APR-02 |
| S-TSK-01 | タスク登録・更新画面 | APPLICANT | F-TSK-01 |
| S-BDG-01 | 予算管理画面 | 全ロール | F-BDG-01 〜 F-BDG-06 |
| S-BDG-02 | 工数実績入力画面 | APPLICANT | F-BDG-01 |
| S-BDG-03 | 直接経費入力画面 | APPLICANT | F-BDG-02 |
| S-NTF-01 | 通知一覧画面 | 全ロール | F-NTF-01 / F-NTF-02 |

## ■ 画面遷移

```text
ログイン画面（S-AUTH-01）
  ↓
ダッシュボード（S-CMN-01）
  ├─ 案件一覧画面（S-APP-01）
  │    ├─ 案件申請画面（S-APP-02）※APPLICANTのみ
  │    └─ 案件詳細画面（S-APP-03）
  │         ├─ 承認・却下画面（S-APR-01）※DEPT_MANAGER / HQ_MANAGERのみ
  │         ├─ タスク登録・更新画面（S-TSK-01）※APPLICANTのみ・承認済み案件のみ
  │         └─ 予算管理画面（S-BDG-01）
  │              ├─ 工数実績入力画面（S-BDG-02）※APPLICANTのみ
  │              └─ 直接経費入力画面（S-BDG-03）※APPLICANTのみ
  └─ 通知一覧画面（S-NTF-01）
```

## ■ 各画面の詳細



---

### S-CMN-01：ダッシュボード（トップ）

**対象ロール**：全ロール

**表示項目**
- 未読通知件数
- 承認待ち件数（DEPT_MANAGER / HQ_MANAGERのみ）
- 自身の案件一覧へのリンク（APPLICANTのみ）
- 部門案件一覧へのリンク（DEPT_MANAGERのみ）
- 全部門案件一覧へのリンク（HQ_MANAGERのみ）
- 通知一覧へのリンク

---

### S-APP-01：案件一覧画面

**対象ロール**：全ロール

**表示項目**

| 項目 | APPLICANT | DEPT_MANAGER | HQ_MANAGER |
|------|-----------|--------------|------------|
| 案件名 | ✅（自身のみ） | ✅（自部門のみ） | ✅（全部門） |
| 担当部門 | ✅ | ✅ | ✅ |
| 申請者名 | ✅ | ✅ | ✅ |
| ステータス | ✅ | ✅ | ✅ |
| 進捗率バー | ✅ | ✅ | ✅ |
| 申請日 | ✅ | ✅ | ✅ |
| 遅延フラグ | ✅ | ✅ | ✅ |

**操作**
- 案件詳細への遷移
- 新規申請ボタン（APPLICANTのみ）

---

### S-APP-02：案件申請画面

**対象ロール**：APPLICANT

**入力項目**

| 項目 | 必須 | 補足 |
|------|------|------|
| 案件名 | ○ | 50字以内 |
| 目的・概要 | ○ | なぜこの案件が必要か、何を実現するか |
| 担当部門 | ○ | 申請者の所属部門が自動セット |
| 概算予算（万円） | ○ | |
| 概算工数（人月） | △ | 任意・入力推奨 |
| 人月単価（万円） | ○ | 案件単位で設定 |
| 開発開始予定日 | △ | |
| 完了予定日 | △ | |

**操作**
- 申請ボタン
- キャンセルボタン

---

### S-APP-03：案件詳細画面

**対象ロール**：全ロール

**表示項目**
- 案件名・目的・概要
- 概算工数・概算予算額・人月単価
- 担当部門・申請者名
- 現在のステータス
- 進捗率バー
- タスク一覧（タスク名・担当者・ステータス・進捗率・開始日・期限・備考）
- 予算サマリ（予算工数・実績工数・工数消化率・予算金額・実績金額・金額消化率・残工数・残予算）

**操作**
- 承認・却下ボタン（DEPT_MANAGER / HQ_MANAGERのみ・承認待ち案件のみ）
- タスク登録・更新ボタン（APPLICANTのみ・承認済み案件のみ）
- 工数実績入力ボタン（APPLICANTのみ・承認済み案件のみ）
- 直接経費入力ボタン（APPLICANTのみ・承認済み案件のみ）

**対象ロール**：DEPT_MANAGER / HQ_MANAGER
