# 予算管理系画面

## 対象画面

- S-BDG-01：予算管理画面
- S-BDG-02：工数実績入力・編集画面
- S-BDG-03：直接経費入力・編集画面
- S-BDG-04：予算管理一覧画面

---

# S-BDG-01：予算管理画面

## ■ 目的

案件の予算・実績を可視化し、消費状況を確認する

---

## ■ 対象ロール

- 全ロール

---

## ■ 業務上の役割

予算消費状況の監視・管理

---

## ■ 表示項目

### 予算サマリ

| 項目       | 説明                                     |
| ---------- | ---------------------------------------- |
| 計画工数   | 計画上の工数（人月）                     |
| 実績工数   | 登録済み工数の合計（人月）               |
| 工数消化率 | 実績工数 ÷ 計画工数 × 100（%）           |
| 残工数     | 計画工数 - 実績工数（人月）              |
| 予算金額   | 予算額（円）                             |
| 実績金額   | 工数実績 × 人月単価 + 直接経費合計（円） |
| 金額消化率 | 実績金額 ÷ 予算金額 × 100（%）           |
| 残予算     | 予算金額 - 実績金額（円）                |

### 工数実績一覧

| 項目             | 説明                            |
| ---------------- | ------------------------------- |
| 対象月           | 工数を入力した月（例：2026-04） |
| 実績工数（人月） | その月の投入工数                |
| 金額（円）       | 実績工数 × 人月単価             |
| 編集・削除ボタン | APPLICANTのみ表示               |

### 直接経費一覧

| 項目             | 説明                                     |
| ---------------- | ---------------------------------------- |
| 発生日           | 経費の発生日                             |
| 経費種別         | 外注費・ライセンス費・機器・備品・その他 |
| 金額（円）       | 経費金額                                 |
| 備考             | 内容説明                                 |
| 編集・削除ボタン | APPLICANTのみ表示                        |

---

## ■ 操作

| 操作         | 内容                   | 条件                            |
| ------------ | ---------------------- | ------------------------------- |
| 工数実績入力 | S-BDG-02（新規）へ遷移 | APPLICANTのみ・承認済み案件のみ |
| 工数実績編集 | S-BDG-02（編集）へ遷移 | APPLICANTのみ                   |
| 工数実績削除 | 確認Dialog後に削除     | APPLICANTのみ                   |
| 直接経費入力 | S-BDG-03（新規）へ遷移 | APPLICANTのみ・承認済み案件のみ |
| 直接経費編集 | S-BDG-03（編集）へ遷移 | APPLICANTのみ                   |
| 直接経費削除 | 確認Dialog後に削除     | APPLICANTのみ                   |

---

## ■ バリデーション / 制御

- ロールに応じて表示範囲を制御
  - APPLICANT：自身の案件のみ
  - DEPT_MANAGER：自部門のみ
  - HQ_MANAGER：全部門
- 承認前ステータス（DRAFT / PENDING_DEPT / PENDING_HQ / REJECTED）の案件では予算API・工数API・経費APIを呼び出さない

---

## ■ 使用API

| メソッド | エンドポイント                               | 説明             |
| -------- | -------------------------------------------- | ---------------- |
| GET      | /projects/{project_id}                       | 案件情報取得     |
| GET      | /projects/{project_id}/budget                | 予算情報取得     |
| GET      | /projects/{project_id}/worklogs              | 工数実績一覧取得 |
| GET      | /projects/{project_id}/expenses              | 直接経費一覧取得 |
| DELETE   | /projects/{project_id}/worklogs/{worklog_id} | 工数実績削除     |
| DELETE   | /projects/{project_id}/expenses/{expense_id} | 直接経費削除     |

---

# S-BDG-02：工数実績入力・編集画面

## ■ 目的

案件への月次投入工数実績を登録・編集する

---

## ■ 対象ロール

- APPLICANT

---

## ■ 業務上の役割

工数実績の月次入力・修正

---

## ■ 入力項目

| 項目             | 必須 | 説明                              |
| ---------------- | ---- | --------------------------------- |
| 対象月           | ○    | 例：2026年4月（編集時は変更不可） |
| 実績工数（人月） | ○    | その月に投入した工数              |

---

## ■ 操作

| 操作       | 内容                       |
| ---------- | -------------------------- |
| 保存       | 工数実績を登録・更新する   |
| キャンセル | 入力を破棄して前画面へ戻る |

---

## ■ バリデーション / 制御

- APPLICANTのみ操作可能
- 承認済み案件（APPROVED / IN_PROGRESS）のみ入力可能
- 同一案件・同一対象月の二重登録は不可
- 実績工数は0以上20以下の値のみ入力可能
- 対象月は未来の月・案件開始日より前の月は選択不可
- 編集時は対象月を変更不可

---

## ■ 画面遷移

- 遷移元：予算管理画面（S-BDG-01）
- 遷移先：予算管理画面（S-BDG-01）

---

## ■ 使用API

| メソッド | エンドポイント                               | 説明                       |
| -------- | -------------------------------------------- | -------------------------- |
| GET      | /projects/{project_id}                       | 案件情報取得               |
| GET      | /projects/{project_id}/worklogs              | 既存工数実績取得（編集時） |
| POST     | /projects/{project_id}/worklogs              | 工数実績登録               |
| PUT      | /projects/{project_id}/worklogs/{worklog_id} | 工数実績更新               |

---

# S-BDG-03：直接経費入力・編集画面

## ■ 目的

案件で発生した直接経費を登録・編集する

---

## ■ 対象ロール

- APPLICANT

---

## ■ 業務上の役割

直接経費の発生都度入力・修正

---

## ■ 入力項目

| 項目       | 必須 | 説明                                     |
| ---------- | ---- | ---------------------------------------- |
| 経費種別   | ○    | 外注費・ライセンス費・機器・備品・その他 |
| 金額（円） | ○    | 999,999,999円以下                        |
| 発生日     | ○    | 未来の日付・案件開始日より前は不可       |
| 備考       | —    | 内容説明                                 |

---

## ■ 操作

| 操作       | 内容                       |
| ---------- | -------------------------- |
| 保存       | 直接経費を登録・更新する   |
| キャンセル | 入力を破棄して前画面へ戻る |

---

## ■ バリデーション / 制御

- APPLICANTのみ操作可能
- 承認済み案件（APPROVED / IN_PROGRESS）のみ入力可能
- 金額は0より大きく999,999,999円以下の値のみ入力可能
- 発生日は未来の日付・案件開始日より前の日付は選択不可

---

## ■ 画面遷移

- 遷移元：予算管理画面（S-BDG-01）
- 遷移先：予算管理画面（S-BDG-01）

---

## ■ 使用API

| メソッド | エンドポイント                               | 説明                   |
| -------- | -------------------------------------------- | ---------------------- |
| GET      | /projects/{project_id}                       | 案件情報取得           |
| GET      | /projects/{project_id}/expenses              | 既存経費取得（編集時） |
| POST     | /projects/{project_id}/expenses              | 直接経費登録           |
| PUT      | /projects/{project_id}/expenses/{expense_id} | 直接経費更新           |

---

# S-BDG-04：予算管理一覧画面

## ■ 目的

全案件の予算消費状況を一覧で確認・管理する

---

## ■ 対象ロール

- DEPT_MANAGER
- HQ_MANAGER

---

## ■ 業務上の役割

部門・全社単位での予算消費状況の監視

---

## ■ 表示項目

| 項目           | 説明                               |
| -------------- | ---------------------------------- |
| 案件名         |                                    |
| 担当部門       | HQ_MANAGERのみ表示                 |
| 予算額         |                                    |
| 実績額         |                                    |
| 予算消化率     | プログレスバー表示                 |
| サマリーカード | 案件数・総予算・総実績・平均消化率 |

---

## ■ フィルター・検索

| 項目                 | 説明                     | 備考                    |
| -------------------- | ------------------------ | ----------------------- |
| キーワード検索       | 案件名の部分一致検索     | デバウンス（300ms）対応 |
| ステータスフィルター | ステータスでの絞り込み   |                         |
| 部門フィルター       | 部門単位での絞り込み     | HQ_MANAGERのみ表示      |
| 予算帯フィルター     | 予算額の範囲での絞り込み |                         |
| 予算列ソート         | 予算額の昇順・降順ソート | サーバーサイドソート    |
| ページネーション     | 1ページあたりの件数切替  |                         |

---

## ■ 操作

| 操作           | 内容                           | 条件 |
| -------------- | ------------------------------ | ---- |
| 案件選択       | 予算管理画面（S-BDG-01）へ遷移 |      |
| フィルター変更 | 検索条件を変更してAPI再取得    |      |

---

## ■ バリデーション / 制御

- DEPT_MANAGER：自部門の案件のみ表示
- HQ_MANAGER：全部門の案件を表示
- サーバーサイドページネーション・フィルター・ソートに対応
- フィルター変更時はページを1にリセット

---

## ■ 使用API

| メソッド | エンドポイント           | 説明                                                     |
| -------- | ------------------------ | -------------------------------------------------------- |
| GET      | /projects                | 案件一覧取得（フィルター・ソート・ページネーション対応） |
| GET      | /projects/budget-summary | 予算サマリー取得（フィルター連動）                       |
| GET      | /departments             | 部門一覧取得（部門フィルター用）                         |
