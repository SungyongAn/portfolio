# 機能一覧

> ユースケースおよびロール権限マトリクスをもとに、
> PoCで実装する機能を課題１に区分して定義します。
> 課題２の機能一覧は `doc/memo/phase2_ideas.md` を参照してください。

---

## ■ 課題１：必須機能一覧

### 認証・ユーザー管理

| 機能ID    | 機能名           | 説明                                    | 対象ロール |
| --------- | ---------------- | --------------------------------------- | ---------- |
| F-AUTH-01 | ログイン         | メール・パスワードによる認証（JWT発行） | 全ロール   |
| F-AUTH-02 | ログアウト       | セッションの終了・トークン破棄          | 全ロール   |
| F-AUTH-03 | トークン自動更新 | アクセストークンの自動リフレッシュ      | 全ロール   |

---

### 案件申請

| 機能ID   | 機能名       | 説明                                                                             | 対象ロール                                         |
| -------- | ------------ | -------------------------------------------------------------------------------- | -------------------------------------------------- |
| F-APP-01 | 新規案件申請 | 案件名・目的・概算予算・工数・開発期間を入力して申請                             | APPLICANT                                          |
| F-APP-02 | 案件一覧確認 | 案件一覧のステータス確認・キーワード検索・ステータスフィルター・ページネーション | 全ロール                                           |
| F-APP-03 | 案件詳細確認 | 申請内容・進捗・予算・タスクの詳細確認                                           | 全ロール                                           |
| F-APP-04 | 案件着手     | 承認済み案件を進行中に移行する                                                   | APPLICANT（自案件のみ）/ DEPT_MANAGER / HQ_MANAGER |
| F-APP-05 | 案件完了     | 進行中案件を完了に移行する                                                       | DEPT_MANAGER / HQ_MANAGER                          |

---

### 承認フロー

| 機能ID   | 機能名         | 説明                                                 | 対象ロール   |
| -------- | -------------- | ---------------------------------------------------- | ------------ |
| F-APR-01 | 一次承認・却下 | 自部門の申請案件を承認または却下（却下理由必須）     | DEPT_MANAGER |
| F-APR-02 | 最終承認・却下 | 一次承認済み案件を最終承認または却下（却下理由必須） | HQ_MANAGER   |
| F-APR-03 | 承認状況確認   | 案件の現在の承認ステータス・却下理由を確認           | 全ロール     |

---

### タスク管理

| 機能ID   | 機能名                 | 説明                                                                           | 対象ロール                             |
| -------- | ---------------------- | ------------------------------------------------------------------------------ | -------------------------------------- |
| F-TSK-01 | タスク登録・削除       | 承認済み案件のタスクを登録・削除する                                           | APPLICANT / DEPT_MANAGER               |
| F-TSK-02 | タスク更新             | タスクのステータス・進捗率を更新する（TASK_MEMBERは自部門案件・進行中まで）    | APPLICANT / DEPT_MANAGER / TASK_MEMBER |
| F-TSK-03 | タスク一覧確認         | 案件配下のタスク一覧・進捗状況を確認する                                       | 全ロール                               |
| F-TSK-04 | 案件進捗率自動算出     | 配下タスクの進捗率の平均で案件全体の進捗率を自動算出する                       | 自動処理                               |
| F-TSK-05 | ステータス・進捗率連動 | ステータスを完了にすると進捗率が100%に、進捗率100%でステータスが完了に自動変更 | 自動処理                               |

---

### 工数・予算管理

| 機能ID   | 機能名                 | 説明                                                                                 | 対象ロール   |
| -------- | ---------------------- | ------------------------------------------------------------------------------------ | ------------ |
| F-BDG-01 | 工数実績入力（月次）   | 案件への月次投入工数実績を入力・編集・削除する                                       | APPLICANT    |
| F-BDG-02 | 直接経費入力           | 外注費・ライセンス費・機材費・その他の直接経費を入力・編集・削除する                 | APPLICANT    |
| F-BDG-03 | 予算サマリ自動算出     | 実績工数・直接経費をもとに実績金額・消化率を自動算出する                             | 自動処理     |
| F-BDG-04 | 予実確認（自案件）     | 自身の案件の予算額・実績額・消化率を確認する                                         | APPLICANT    |
| F-BDG-05 | 予算管理一覧（自部門） | 自部門の全案件の予算消費状況・サマリーを確認する（フィルター・ページネーション対応） | DEPT_MANAGER |
| F-BDG-06 | 予算管理一覧（全部門） | 全部門の全案件の予算消費状況・サマリーを確認する（フィルター・ページネーション対応） | HQ_MANAGER   |
| F-BDG-07 | 予算サマリー集計       | 案件数・総予算・総実績・加重平均消化率をSQL集計で算出する                            | 自動処理     |

---

### 案件一覧・進捗確認

| 機能ID   | 機能名                 | 説明                                                     | 対象ロール   |
| -------- | ---------------------- | -------------------------------------------------------- | ------------ |
| F-PRG-01 | 案件一覧確認（自部門） | 自部門の全案件の一覧・進捗率を確認する                   | DEPT_MANAGER |
| F-PRG-02 | 案件一覧確認（全部門） | 全部門の全案件の一覧・進捗率を確認する                   | HQ_MANAGER   |
| F-PRG-03 | ダッシュボード         | 案件進捗・予算消化状況・未読通知数をカード形式で確認する | 全ロール     |

---

### 通知

| 機能ID   | 機能名                   | 説明                                                     | 対象ロール |
| -------- | ------------------------ | -------------------------------------------------------- | ---------- |
| F-NTF-01 | アプリ内通知             | 承認依頼・承認完了・却下・着手・完了をアプリ内で通知する | 全ロール   |
| F-NTF-02 | リアルタイム通知         | WebSocketによるリアルタイムプッシュ通知・トースト表示    | 全ロール   |
| F-NTF-03 | 通知既読管理             | 通知の個別既読・全件既読を管理する                       | 全ロール   |
| F-NTF-04 | 通知一覧ページネーション | 通知一覧のページネーション表示                           | 全ロール   |
