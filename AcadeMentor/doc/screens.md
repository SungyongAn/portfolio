## 2. 画面設計詳細

### 2-1. ログイン／新規登録画面

**レイアウト**：左右2カラム
- 左パネル：AcadeMentorの特徴紹介（紺色背景）
- 右パネル：ログイン／新規登録フォーム

**仕様**

| 項目 | 内容 |
|---|---|
| 入力項目 | メールアドレス・パスワード（新規登録時は利用者種別も選択） |
| 利用者種別 | 保護者 / 子どもをボタンで選択。ログイン後の遷移先が変わる |
| バリデーション | メール形式チェック・パスワード8文字以上・空欄チェック |
| エラー表示 | 認証失敗時はメッセージ＋入力欄を赤枠でハイライト |
| パスワード表示 | 目アイコンで表示／非表示の切り替え |
| ログイン後遷移 | 保護者 → 進捗ダッシュボード　子ども → ホーム |

### 2-2. ホーム画面（子ども向け）

**レイアウト**：左サイドバー＋メインコンテンツ（2カラム）

**仕様**

| 項目 | 内容 |
|---|---|
| 表示情報 | 今日の正答率・回答数・連続学習日数・累計問題数（学習時間は非表示） |
| おまかせ演習 | AIが弱点・過去問傾向をもとに問題種別を自動選択して出題 |
| 種別選択演習 | 計算・図形・文章・特殊算の4種別から選んで出題。各ボタンに正答率とプログレスバーを表示 |
| 正答率の色分け | 70%以上：緑　70%未満：赤で一目で得意・苦手を判断できる |
| 苦手単元 | 正答率の低い単元を最大3件表示。クリックで該当種別の演習に遷移 |
| 連続学習 | 学習済み時は連続学習日数をバッジ形式で強調表示 |
| ナビゲーション | 左サイドバーにホーム・問題演習・弱点進捗の3項目 |

**補足**
- 学習時間は画面を開いているだけでカウントされる問題があるため子ども向け画面には表示しない
- 未学習時と学習済み時でメインボタンの表示が変わる（「今日の問題に挑戦する」→「追加で問題を解く」）

### 2-3. 問題演習画面

**レイアウト**
- 演習開始前：中央寄せ（問題数選択UI）
- 演習中：上段に問題文（全幅）、下段に今回の進捗と累計正答率を横並び

**問題形式一覧**

| 問題形式 | 回答形式 | 採点方式 | 備考 |
|---|---|---|---|
| 計算問題 | 数値入力 | 完全一致で自動採点 | 四則演算・分数・小数など |
| 図形問題 | 数値入力＋単位 | 完全一致で自動採点 | 図はテキスト説明で代用（MVP） |
| 文章問題 | 数値入力＋単位 | 完全一致で自動採点 | 速さ・割合・比・場合の数など |
| 特殊算 | 数値入力＋単位 | 完全一致で自動採点 | つるかめ算・植木算・旅人算など |

**仕様**

| 項目 | 内容 |
|---|---|
| 演習開始前 | おまかせ：確認画面のみ。種別選択：5・10・15・20問から選択して開始 |
| おまかせの問題数 | 固定数。過去問入手後に決定 |
| 進捗バー | 画面上部に現在の問題数／総問題数を表示。回答するごとに進む |
| 下段左：今回の進捗 | 正解数・不正解数・正答率・○×履歴ドットを表示 |
| 下段右：累計正答率 | 4種別の正答率を常時表示 |
| ヒント機能 | なし |
| 採点結果（正解） | 緑で正解を表示 |
| 採点結果（不正解） | 赤で自分の答えと正答を並べて表示 |
| 採点後の遷移 | 「解説を見る」→ 解説画面へ　「次の問題」→ 次の問題へ |

### 2-4. 解説画面

**レイアウト**：3段構成（全幅）

| 段 | 内容 |
|---|---|
| 上段 | 問題文・採点結果（正解・不正解・正答）を全幅で表示 |
| 中段 | AIが生成するステップ解説を縦並びで表示。ステップ間を線でつなぐ |
| 下段 | この問題のポイント（AIが考え方・注意点をまとめて表示） |

**仕様**

| 項目 | 内容 |
|---|---|
| ステップ解説 | 番号・タイトル・説明文・計算式を縦並びで表示。ステップ間を線でつなぐ |
| ポイント | AIがこの問題で押さえるべき考え方・注意点を全幅で表示 |
| 正解時 | 緑で正解を表示 |
| 不正解時 | 自分の答えと正答を並べて表示 |
| アクション | 「もう一度解く」で同じ問題を再挑戦。「次の問題へ」で演習を継続 |

### 2-5. 弱点・進捗画面（子ども向け）

**レイアウト**：サマリー → 2カラム（種別正答率＋学習カレンダー） → 単元別テーブル

**仕様**

| 項目 | 内容 |
|---|---|
| サマリー | 累計回答数・総合正答率・連続学習日数・苦手単元数を4列で表示 |
| 問題種別正答率 | 4種別をバーグラフで色分け表示（70%以上：緑・60〜70%：黄・60%未満：赤） |
| 学習カレンダー | 過去4週間の学習量を色の濃さで可視化 |
| 単元別正答率 | 正答率の低い順に表示。推移（先週比）・回答数・集中練習ボタンを併記 |
| 集中練習ボタン | クリックするとその単元に絞った問題演習画面に遷移 |

**単元について**
- 単元とは問題種別をさらに細かく分けた分類（例：複合図形の面積・旅人算・速さ・時間・距離など）
- 単元の分類はAIが過去問を分析した際に自動で設定する
- 志望校によって頻出単元が異なるため、過去問アップロード後にAIが単元を定義する
- 苦手単元数 = 正答率70%未満の単元数

### 2-6. 進捗ダッシュボード（保護者向け）

**レイアウト**：注目アラート → サマリー → 2カラム（週間学習時間＋画面別滞在時間） → 2カラム（種別正答率＋苦手単元）

**仕様**

| 項目 | 内容 |
|---|---|
| 期間切替 | 今週・今月・全期間の3つで表示データを切り替え |
| 注目アラート | 正答率が下がった単元など保護者が注目すべき変化をAIが自動検知して表示 |
| サマリー | 学習時間・回答数・正答率・連続学習日数。先月比の増減を表示 |
| 週間学習時間 | 曜日別の棒グラフ。今週の合計・1問あたりの時間・回答数を表示 |
| 画面別滞在時間 | 問題演習・解説・弱点進捗・ホームの4画面の滞在時間をバーグラフで表示 |
| 問題種別正答率 | 4種別の累計正答率をバーグラフで色分け表示 |
| 苦手単元 | 正答率70%未満の単元を一覧表示。先週比の推移を併記 |
| 子ども切替 | 複数子どもを登録している場合は切替メニューで子どもを選択できる |

**子ども向け画面との主な違い**
- 注目アラートでAIが自動的に保護者へ変化を通知
- 週間学習時間グラフで学習習慣の把握が可能
- 1問あたりの時間で学習密度が把握できる
- 画面別滞在時間でどこに時間を使っているかが把握できる
- 期間切替で今週・今月・全期間の比較が可能

**学習時間の計測方針**
- 問題表示から回答送信までの時間を積算（画面を開いているだけはカウントしない）
- 画面別滞在時間：各画面への遷移・離脱のタイムスタンプをバックエンドで記録
- 異常値の除外：一定時間（例：10分以上）回答がない場合はカウントを停止

### 2-7. 過去問アップロード（保護者向け）

**レイアウト**：左右2カラム（左：アップロードフォーム　右：アップロード済みリスト）

**仕様**

| 項目 | 内容 |
|---|---|
| 種別選択 | 計算・図形・文章・特殊算の4種別から選択してからアップロード |
| 年度入力 | 試験年度を選択。複数年度の過去問を管理できる |
| ファイル形式 | PDF・PNG・JPG対応。最大50MB。ドラッグ＆ドロップまたはファイル選択 |
| AI分析結果 | 分析完了後に出題傾向・単元・難易度をタグ形式で表示 |
| ステータス管理 | 分析中・分析完了をバッジで表示 |
| 削除機能 | アップロード済みファイルの削除が可能 |
| 著作権注意 | アップロード前に著作権への配慮について毎回明示 |

**著作権への配慮**
- アップロードされた過去問はAIが出題傾向・形式・難易度の分析にのみ使用
- 問題文や解答をそのままデータベースに保存・表示しない
