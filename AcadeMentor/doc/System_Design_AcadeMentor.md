## 1. 設計方針

| 項目 | 内容 |
|---|---|
| 対象プラットフォーム | Webブラウザ（PC限定） |
| 対象教科 | 算数のみ（MVP） |
| 問題種別 | 計算問題・図形問題・文章問題・特殊算の4種別 |
| フロントエンド | Next.js（React）・Tailwind CSS → Vercel |
| バックエンド | Python・FastAPI・Anthropic Python SDK → Oracle Cloud |
| データベース | PostgreSQL・SQLAlchemy → Oracle Cloud |
| 認証 | JWT認証（FastAPI実装） |
| AI | Claude API（問題生成・採点・解説・過去問分析） |
